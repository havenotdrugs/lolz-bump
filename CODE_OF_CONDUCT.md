# Code of Conduct

Этот проект придерживается базовых принципов уважительного и профессионального взаимодействия.

## Ожидаемое поведение

- Уважение к участникам сообщества.
- Конструктивная обратная связь.
- Обсуждение идей, а не личностей.

## Недопустимо

- Оскорбления, травля, угрозы.
- Публикация чувствительных данных без согласия.
- Дискриминация в любой форме.

## Контакты

При проблемах создайте приватное обращение через Security Policy или обратитесь к мейнтейнерам проекта.
