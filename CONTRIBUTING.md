# Contributing

Спасибо за вклад в `lolz-bump`.

## Быстрый старт

```bash
python3 -m venv .venv
source .venv/bin/activate
pip install -e .[dev]
pytest -q
```

## Правила

- Делайте небольшие и атомарные PR.
- Перед PR запускайте `pytest -q`.
- Не коммитьте секреты (`.env`, токены, ключи).
- Для изменений поведения добавляйте/обновляйте тесты.

## Коммиты

Рекомендуемый формат:

- `feat: ...`
- `fix: ...`
- `docs: ...`
- `test: ...`
- `chore: ...`

## Pull Request

В PR опишите:

- что изменилось;
- зачем это нужно;
- как проверяли (команды/результат).
