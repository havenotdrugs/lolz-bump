# Security Policy

## Поддерживаемые версии

На данный момент поддерживается только актуальная ветка `main`.

## Как сообщить об уязвимости

- Не публикуйте уязвимость в открытом issue.
- Сообщите приватно мейнтейнерам проекта.
- Приложите шаги воспроизведения и ожидаемое влияние.

После подтверждения проблемы будет подготовлен фикс и опубликовано уведомление.
